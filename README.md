Um jogo bastante famoso intitulado de "Jogo da Cobrinha".
Feito na linguagem Python, o jogo teve o intuito do estudo mais aprofundado da biblioteca <b>PyGame</b> e da <b>Lógica para jogos</b> feitos em Python.
